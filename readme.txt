Display your BuddyPress site wide posts in a paginated widget

== Description ==
If you're using BuddyPress and displaying site wide posts, this plugin is a necessary replacement for the one that comes packaged with BuddyPress. The widget options include title, posts per page, total posts, number of posts with excerpt, and previous and next link titles.

== Installation ==
1. Upload `bp-paginated-posts.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the 'Widgets' menu and drag the 'Paginated Site Wide Posts' widget to your sidebar and input your options.
